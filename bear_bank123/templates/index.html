Bem-vindo ao Bear Bank

Este é um aplicativo de demonstração desenvolvido como parte de um Trabalho de Conclusão de Curso (TCC) no Instituto Federal de Brasília.

O objetivo principal deste projeto é mostrar e ensinar sobre as vulnerabilidades de SQL Injection e as melhores práticas para prevenir esses ataques em aplicações web.

O Bear Bank oferece dois modos para fins educacionais:

  • Modo Vulnerável: Demonstra como as vulnerabilidades de SQL Injection podem ser exploradas.
  • Modo Seguro: Implementa medidas de segurança, como validação de entradas, prepared statements e sanitização de dados para prevenir SQL Injection.

Sinta-se à vontade para explorar o aplicativo e ver o impacto das práticas de segurança em ação.

Por favor, faça login para acessar sua conta e visualizar a demonstração em ação.
